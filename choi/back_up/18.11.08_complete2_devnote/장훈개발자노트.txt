맡은 곳 : header, title, footer

첫 과제 결과물인 html_hw.html파일을 이용 해서 각자의 영역에 맞는 html태그들을 추출해서 작업을 했습니다.
'검사'로 html 엘리먼트들과 css 조합을 통해 만들었습니다.

한 개의skeleton을 두고 공유 한 것이아니라, 각자의 폴더에 서로 맡은 영역만 작업을 하는 방식 이므로
PR에서 코드 충돌이 단 한번도 발생하지 않았습니다.
그러나 서로 레이아웃 설정 방식이 모두 달라서
병합 할 때 레이아웃 위치 잡는 것이 난해했습니다.

개선방안, 아쉬운 점
-레이아웃 설계시, 한 영역을 모두 감싸서 해당 영역의 요소들이 모두 움직이게 염두에 두는 것
-복잡한 코드 설계시, 지금의 작업 방식으로는 병합할 때 더 어려울 수도 있는 것
-플렉스에 대한 이해가 있었다면, 레이아웃 설정이 더 쉬웠을 것 같다.
